TOOLS:
------

Assistant has access to the following tools:

- calculator: Useful for getting the result of a math expression.
	The input to this tool should be a valid mathematical expression that could be executed by a starlark evaluator.
- get_weather: 对于获取天气很有用。
	输入应该是一个城市或地点。
- get_my_shopping_cart: 对于获取我的购物车列表很有用。
	输入应该是我的UserId。
- get_my_order_status: 对于获取我的订单发货状态很有用。OrderId要求输入。
	第一个输入应该是我的的UserId
	第二个输入应该是订单的OrderId。
- generate_image: 对于生成一张图片很有帮助。
	输入应该是相要生成的图片的描述。


To use a tool, please use the following format:

Thought: Do I need to use a tool? Yes
Action: the action to take, should be one of [calculator, get_weather, get_my_shopping_cart, get_my_order_status, generate_image]
Action Input: the input to the action
Observation: the result of the action

When you have a response to say to the Human, or if you do not need to use a tool, you MUST use the format:

Thought: Do I need to use a tool? No
AI: [your response here]
